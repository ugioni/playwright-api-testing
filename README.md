# Como começar

### Playwright docs:
 - [Playwright](https://playwright.dev/java/docs/api-testing)

### Pré requisitos:
 - Java JDK 8

### Como executar?

Dentro do painel do Maven:

 1. `mvn clean`
 1. `mvn test`

### Allure reports - Relatório HTML

- comando: mvn allure:report
- pasta target >> site >> index.html
